## 🧰 Hardware Used
- **Microcontroller:** ESP32 Development Board (e.g., WROOM-32)
- **Components:** LEDs, 220Ω/330Ω Resistors, Push Buttons, Active Buzzer, Joystick, Potentiometer
- **Prototyping:** Breadboard and Jumper Wires
